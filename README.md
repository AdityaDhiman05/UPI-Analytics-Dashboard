## 📌 Project Overview
An end-to-end data science project analyzing Indian UPI transaction 
data to uncover growth trends, build predictive models, and serve 
insights through an interactive Streamlit dashboard.

**Key Results:**
- Trained Random Forest model to predict UPI transaction volumes
- Built interactive Streamlit dashboard with actual vs predicted trends
- Analyzed 10 structured notebooks covering full data science lifecycle

---

## 🎯 Objectives
- Explore and visualize UPI transaction growth trends in India
- Perform feature engineering and correlation analysis
- Predict future UPI transaction volumes using ML
- Deploy an interactive dashboard for real-time insights

## 🤖 Machine Learning
- **Model:** Random Forest Regressor
- **Task:** Predict UPI transaction volumes
- **Output:** Exported deployable model → `best_random_forest_model.pkl`
- **Features:** Growth rate, month, lag features, correlation-selected variables

## 📊 Dashboard Features
Built with **Streamlit** (`app.py`):
- Monthly UPI transaction volume visualization
- Actual vs Predicted volume comparison
